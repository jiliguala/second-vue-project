# web 前台门户网站


## 安装必要的依赖
```
npm i element-plus axios
```
## 安装页面加载进度条
```
npm install --save nprogress
```
## 安装moment
```
npm i moment --save
```
## 安装lodash工具库，内部封装了字符串、数组、对象等常见数据类型的处理函数
```
npm i lodash --save
```


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
